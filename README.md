# 🚀 **Калькулятор обоев**
Добро пожаловать в репозиторий проекта калькулятора обоев! Здесь вы найдете функциональный калькулятор для расчета необходимого количества обоев для вашего пространства.

🖥 [Посмотреть калькулятор](#)

## 📌 **О проекте**
Этот проект представляет собой калькулятор для расчета обоев, созданный с использованием Vite, React и TypeScript. В нем реализованы:

- 🔹 Главная страница с функционалом калькулятора раппорта, размера обоев с учетом дверей и окон
- 🔹 Форма для ввода параметров комнаты и обоев
- 🔹 Визуализация результатов расчета
- 🔹 Простая и удобная навигация
- 🔹 Поддержка типов через TypeScript
- 🔹 Анимации и уведомления с использованием React Hook Toast
- 🔹 Валидация данных с помощью Zod

## 🛠 **Используемые технологии**
- **Vite** — сборщик для быстрого старта проектов
- **React** — библиотека для создания пользовательских интерфейсов
- **TypeScript** — строгая типизация кода
- **React Hook Form** — управление формами
- **Zod** — валидация данных
- **React Hot Toast** — уведомления
- **@hookform/resolvers** — интеграция с Zod для валидации форм

## 🔧 **Запуск проекта**
1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/Licrua/portfolio.git
   ```
2. Установите зависимости:
   ```sh
   npm install
   ```
3. Запустите локальный сервер:
   ```sh
   npm run dev
   ```
